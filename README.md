# TPS-Shooter-Game
This (basic) game is made with Unreal Engine 4.27

Features:
+ Drop/Equip Weapons ✔️
+ Sprint/Walk/Crounch ✔️
+ Switch weapons 🔃
+ Inventory 🔃
+ Reload system ✔️
+ Fire Mode (Single-Auto) 🔃
+ Ragdoll Recover (without animation) ✔️
+ Head Rotation ✔️
+ To be continued... 🔃
